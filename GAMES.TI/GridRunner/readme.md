# Grid Runner

This demo was 1st developed by SteveB@atariage.com, to show scrolling the screen to the left. Stephan is famous for the creation of a structured BASIC transpiler TICoded, that takes source code in a BASIC dialect that he developed and it outputs a TI Extended BASIC program that can be compile by the BASIC compiler.

https://lizardware.de/

Later BradSnyder added some features.

I took Brad's version of the BASIC program and translated it to Camel99 Forth and added some colour and sound. The original BASIC version makes use of some excellent library calls available in the TI-99 game development system and compiler which include windowing and scrolling the screen left.  I have neither of those feature so we had to write a way to scroll a portion of the screen using Forth.

So we have yet another example of how to write a simple game with Camel99 Forth.
