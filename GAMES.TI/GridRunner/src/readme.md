# Grid Runner

This demo was 1st developed by SteveB@atariage.com.
Stephan is famous for the creation of a structure BASIC transpiler TICoded,
that takes a BASIC dialect that he developed and outputs an TI Extended BASIC program.

https://lizardware.de/

Later BradSnyder added some features.

I took the BASIC program and translated it to Camel99 Forth and added a some more colour and sound. It shows how to write a simple game with Camel99 Forth.
